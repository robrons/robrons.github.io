Work

My work consisted of building a comprehensive user management system for a selective organisation. The system that I built is a web application that supports features such as an automated mailer, application forms, feedback logger and user statistics. Since, the privacy of the users of this application is protected by FERPA, I have also built into this application security features such as JSON Web Tokens and utilization of single sign-on protocol. This application was developed using full-stack methodologies with the use of MERN Stack.
